How to publish on GitHub Pages

  1. Create a repository, for example koppen-app.
  2. Save this file as index.html.
  3. Push it to the repository root.
  4. In GitHub: Settings → Pages.
  5. Set source to Deploy from a branch.
  6. Select branch main and folder /root.
  7. Your app will be published automatically.

If you want the app on your main site, place this file in a subfolder and link to it from your homepage.
